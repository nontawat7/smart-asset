ยืนยันการดำเนินงาน?

ต้องการดำเนินการขั้นตอนถัดไปหรือไม่

ตั้งค่าเชื่อมโยงระบบฐานข้อมูล Firebase Cloud

วิธีเริ่มการเชื่อมโยงข้อมูลคลาวด์:

  1. เปิดบัญชี Firebase Console และสร้าง Project ใหม่
  2. เปิดใช้งานเมนู Firestore Database (เลือก Start in Test Mode)
  3. กดที่ไอคอนเฟือง ⚙️ > Project Settings และก๊อปปี้ Firebase SDK Configuration มากรอกด้านล่าง

ตรวจพบข้อมูลครุภัณฑ์ภายในเครื่อง!

ต้องการอัปโหลดชุดพัสดุจากเครื่อง ขึ้นคลาวด์ Firebase ตอนนี้เลยหรือไม่?
